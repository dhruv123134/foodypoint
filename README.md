🍔 FoodyPoint - Food Ordering System

📌 Description

FoodyPoint is a full-stack web-based food ordering system that allows users to browse food items, add them to cart, and place orders. It also includes an admin panel to manage products and orders.

---

🚀 Features

👤 User Side

* User Registration & Login
* Browse Food Menu
* Add to Cart
* Place Orders

🛠 Admin Side

* Add / Update / Delete Food Items
* Manage Orders
* View Customer Details

---

🛠 Tech Stack

* HTML, CSS, JavaScript
* PHP
* MySQL
